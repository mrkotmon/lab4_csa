# Схемы

Схемы архитектуры процессора в формате SVG (открываются в браузере и
рендерятся прямо на GitHub). Выполнены в академическом стиле «тракт
данных + устройство управления», как принято в курсе.

- [`datapath.svg`](datapath.svg) — тракт данных (DataPath):
  регистровый файл (R0–R7 + векторные V0–V3), АЛУ (скалярное и
  векторное), память (фон-Нейман), флаги, memory-mapped I/O и шины
  между ними. Управляющие сигналы показаны пунктиром.

- [`control_unit.svg`](control_unit.svg) — устройство управления
  (Control Unit). Поскольку вариант — **hardwired** (`hw`), управление
  построено на **комбинационном дешифраторе** (`тип = opcode >> 5`)
  и **счётчике стадий** (FETCH → DECODE → EXECUTE → MEMORY →
  WRITEBACK → CHECK_IRQ), без микропрограммного ПЗУ. Отдельно показана
  логика прерываний (вариант `trap`).

Те же схемы продублированы в корневом `README.md` в виде Mermaid-диаграмм.
